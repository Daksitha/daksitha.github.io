MeLaX: Conversations with Generative AI

About MeLaX

MeLaX is a real-time platform integrating generative AI technologies within interactive, photorealistic virtual agents.

It enables dynamic, emotion-aware interactions through a modular pipeline, including speech recognition (ASR), natural language processing (LLM), text-to-speech (TTS), and behavior generation.

Features

  • Modular pipeline for ASR, LLM, TTS, and behavior generation
  • No-code GUI for easy experimentation
  • Emotion-aware agent responses
  • Supports custom model configurations
